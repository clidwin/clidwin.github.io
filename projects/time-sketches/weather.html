How might we visualize the weather?

I re-present aspects of weather in an ambient display instead of simply showing a picture or video stream of the outdoors.
View Visualization

Client

This school project from Fall 2014 was created as part of a course on Processing. The topic for the assignment was to use Processing in creating a visual piece, and I developed the concept of using real-time weather data as the foundation.

Inspiration

This weather visualization is inspired by spaces (especially workplaces) without natural lighting. I wanted to find a way to display the temperature and wind speed of my local area with more than plain text or literal imagery.

Languages and Technologies

This weather visualization utilizes the following technologies:
HTML5 & CSS3
p5.js
Weather API

How It Works

This visualization is built using the Processing's javascript library and the temperature and wind speed data points from Weather Underground. Every fifteen minutes, the visualization requests updated weather information from Weather Underground. Updated information from Weather Underground changes the color of the visualization based on the temperature and the speed of triangles based on the wind speed.

< 0°F
1°F - 16°F
17°F - 32°F
33°F - 49°F
50°F - 67°F
68°F - 85°F
> 85°F

While this is meant to be an ambient piece, people can interact with the visualization by clicking on the city and state name indicated in the bottom right hand corner of the visualization. Users are then prompted to enter a US city and state to update the visualization with a different location than the default Blacksburg, VA.
