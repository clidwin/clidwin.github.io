How might we learn about web technologies?

View Website

Client

This project began as a course-long study on the topics of web development software, CSS transforms, and CSS3 animations. It was built with the intention on being a framework to build upon with future tutorials.

Inspiration

I have spent a lot of time learning new technologies for work, school, and personal projects, so I wanted to combine what I had learned into a more comprehensive resource. I also wanted something both computer scientists and artists could use.

How It Works

Each module in the web tutorials focuses on a concept of web technologies and is divided into six parts. The first is an overview of the topic and what it provides to web development as a whole, and the middle four sections explore more specifically the syntax and implementation practices. The last section of each module includes a list of resources to augment these tutorials.

Each section, where applicable, also contains CodePens that provide sample code and small-scale challenges based on the material covered. It is an opportunity to try out each concept before applying it to other projects.
