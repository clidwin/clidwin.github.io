CSS Animations: References

CSS3 Animation Cheat Sheet is a fun and interactive way to see a small sample of possible CSS3 animations. The site also has a downloadable CSS file that can be used to apply these animations to your own site.
W3C: CSS Animations is the official documentation on CSS Animations. The provider, W3C, is the World Wide Web Consortium, which is the group that develops standards for the web. It is a more technical source than some of the other sites, but is a great resource to see what the current standards are and what future standards are being drafted.
Mozilla Developer Network (MDN) is essentially a Wikipedia for web technologies. It is useful for looking up what values a property can have, but what I find especially valuable are its up-to-date listings of what browsers support a certain property.
Keyframe Animation Syntax is a code snippet within the CSS-Tricks website which goes over the basics of CSS3 Animations. This post provides a different way for understanding CSS3 animations by discussing different syntaxes, and the site as a whole is a great resource for both general topics as well as examples of commonly created elements.
Comparing the effectiveness of different educational uses of program animations is an article that outlines a study conducted to see how effective animations are as educational tools. It is a technical journal article related to this topic in that the findings provide insight into when animations are useful for educational contexts.
