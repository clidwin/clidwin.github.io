CSS Animations: Animation Shorthand

The various animation sub-properties can either be used as detailed in the previous section, or combined under the single animation property (with vendor prefixes, as needed). This relationship is just like how the background property encapsulates background-color background-size, etc.

Animation

The animation property accepts all of the animation-subproperties, and will assume defaults for sub-properties that are not specified as values. The only ordering that matters is that duration must precede delay (if both sub-properties are specified). By convention, most people start with the animation-name, though, to help with clarity.

As an example, if someone wanted to create an animation called bounce that lasted 2 seconds and eased in both directions, they could use: animation: bounce 2s ease;
