CSS Transforms: Rotate

The rotate() function spins an element around its center. The function takes in one argument, the number of degrees to rotate, and that value must have the characters deg after it, to indicate the type of value is degrees.

As an example, if someone wanted to rotate a square 45 degrees, they would define in their css transform: rotate(45deg).
