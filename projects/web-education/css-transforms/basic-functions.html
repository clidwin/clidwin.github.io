CSS Transforms: Basic Functions

The values of the CSS transform property are functions written in the form of a keyword, followed by parentheses enclosing values that help specify how to apply the transform. This syntax is similar to the function url(), which can be used to specify a background image, import a font, or otherwise link to a resource that cannot be created inside the CSS stylesheet.

The basic CSS transform functions include rotate, scale, skew, and translate.

Rotate

The rotate() function moves an element around a point in space (if unspecified, the center). It takes in one argument, the number of degrees to rotate, and that value must have a unit specified with it (such as deg) to specify the input type.

As an example, if someone wanted to rotate a square 45 degrees, they could define this in their css as transform: rotate(45deg);.

Scale

The scale() function changes the size of an element. To uniformly scale the element, use one number as the single argument to the function. To change the horizontal and vertical scaling, use two arguments, where the first argument is the value of the horizontal scaling, and the second argument is the value of the vertical scaling. In both forms of this function (one or two arguments), the number can be positive or negative and can be either an integer or decimal.

As an example, if someone wanted a square to be half its normal size, they could define this in their css as transform: scale(0.5);.

Skew

The skew() function changes the element's boundary angles. To skew the entire element at the same angle, use one value as the single argument to the function. To change the horizontal and vertical skew, use two arguments, where the first argument is the angle of the horizontal skew, and the second argument is the angle of the vertical skew. In all cases, the value must also have a unit specified with it (such as deg) to specify the input type.

Angle Units

While degrees are one type of input into the rotate and skew functions, these functions accepts any of the angle data types as possible values.

As an example, if someone wanted to skew a square 30 degrees, they could define this in their css as transform: skew(30deg);.

Translate

The translate() function changes the drawn position. To translate in the x direction, use one argument. To translate horizontally and vertically, use two arguments, where the first argument is the angle of the horizontal shift, and the second argument is the angle of the vertical shift. In all cases, the value must also have a unit specified with it (such as px) to specify length type being referenced.

Length Units

Pixels are one type of input into the translate function, but this function accepts any of the length data types as possible values.

As an example, if someone wanted to draw a square 25px to the right of its origin, they could define this in their css as the transform transform: translate(25px);.

Transforms in One Dimension

In all of the transforms we've talked about so far (rotate, scale, skew, translate), there are functions with similar syntax that apply the transformation in one dimension. To use these values for transformations, add a capitalized character indicating the dimension we want to apply the transform to the function name. This then changes our inputs to this function so that it only takes a single argument, the value for the transform.

This means that if we wanted to just scale a square in the y direction, we could do so by using the CSS transform: scaleY(1.5);.
