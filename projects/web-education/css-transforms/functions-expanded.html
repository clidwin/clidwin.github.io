CSS Transforms: Functions Expanded

Rotate, scale, skew, and translate apply several different types of transformations to an element on their own, but Another CSS transform property is the matrix function, which combines rotate, translate, scale, and transform into a single function that relies on matrix multiplication and mathematics.

Transforms in One Dimension

In all of the transforms we've talked about so far (rotate, scale, skew, translate), we can also append to the function name a capitalized character indicating the dimension we want to apply the transform to. This then changes our inputs to this function so that it only takes a single argument, the value for the transform.

This means that if we wanted to just scale a square in the y direction, we could do so by using the CSS transform: scaleY(1.5);.

matrix()

The matrix function uses a 2d matrix to apply transforms to an element. The first four arguments are the first two rows of the matrix, and are numbers (integers or decimals) that rotate, scale, and skew the element. The last two arguments are the horizontal and vertical translations. Together, all of these arguments satisfy six of the nine positions of a 3 by 3 matrix, which can be used to support 3d transformations[1].

As an example, if someone wanted to scale a square and rotate it 45 degrees, they might use the CSS transform: matrix(1, -1, 1, 1, 20, 20);.As an alternative, they could use the rotate, translate, and scale functions all listed after the transform property.
