Module 2: CSS Transforms

This section builds on the principles of basic CSS and delves into the concept of transforms. By using this property, web designers and developers are able to perform alterations such as translations, rotations, skews, and perspectives on web objects. These effects not only apply to static content, but also provide visual interest in CSS3 animations.

The blue square above demonstrates some of the transform options that will be discussed in this module. When hovering on this square, an animation plays showing the transforms of rotate, scale, and translate applied on a two second interval.

Overview

A transform in CSS is a property that changes the way an element interacts with the grid structure of the DOM (Document Object Model, or element hierarchy). The property affects the element similar to how the Free Transform tool affects a group in Photoshop, manipulating the element/group as a whole instead of each of its contents/layers individually.

The values for the transform property are the functions matrix, rotate, scale, skew, and translate There are also sub-properties of transform that allow us to specify the origin, style, and perspective of the transform.
