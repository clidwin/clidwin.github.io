Module 2: CSS Transforms

This section builds on the principles of basic CSS and delves into the concept of transforms, a CSS3 property that allows web designers and developers to perform alterations such as translations, rotations, skews, and perspectives. Transforms not only provide these capabilities to static content, but also can be used to provide visual interest in CSS3 animations (which will be further discussed in the next module).

The blue square demonstrates some of the transforms that will be discussed in this module. By hovering on this square, an animation will play showing the transforms of rotate, scale, and translate are applied on a two second interval.

Overview

A transform in CSS is a property that changes the way an element interacts with the grid structure of the DOM (Document Object Model, or element hierarchy). The property affects the element similar to the Marquee tools in Photoshop, manipulating the element as a whole instead of each of its contents individually.

The values for the transform property are the functions matrix, rotate, scale, skew, and translate. There are also sub-properties of transform that allow us to specify the origin, style, and perspective of the transform.
