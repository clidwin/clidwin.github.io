Design Tools: Brackets

Writing code is a relatively old process in web design, and because of its roots in the industry there are many tools out there to write code, expanding everything from text editors to graphical drag-and-drop interfaces. Brackets leans more toward the text editor side, but augments the web development experience by providing useful features such as autocomplete common CSS characteristics and closing HTML tags. It also integrates with a browser to give live previews of elements, providing a way to interact with code and design at the same time.

Brackets is an Adobe open source product available for beta download at http://brackets.io/

Program Highlights

Brackets as a web development tool:

  • Creates a strong connection between the code and visual components through Live Preview.
  • Provides syntax styling for many web and non-web native technologies including HTML, CSS, Javascript, Java, and C++.
  • Allows for CSS information to be extracted from PSD files.
  • Does not provide a non-coding solution to web design.
  • Connection between Brackets and the browser is broken if an element is inspected with the browser developer tools.
  • All files open in brackets are listed in the same category, which makes it difficult to switch between projects or file types.

Recommendation

By comparison to its competitor Sublime Text, Brackets shines in its ability to integrate with a live browser preview. It is a great program for anyone who writes code for the web, especially with the syntax highlighting and suggested autocompleting capabilities.
