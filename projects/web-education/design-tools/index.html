Module 1: Design Tools

As technology advances and people populate the internet with a variety of content, a common question for entrepreneurs and aspiring content creators is "how do people create media for digital platforms?" But before we can tackle this question, we first need to figure out what type of media is being created. In some cases, this media is a website, while for others it's an application, and still other cases want either both forms or something completely different.

Learning about interface development tools provides insight to how people in both the industry and academia are developing content for the web. In this section, we explore a few of these tools, each of which offers a unique set of features and properties meant to assist in different stages of the development processes, ranging from prototyping to implmenetation.

Overview

With countless options for what type of media to create, it is no surprise that there are also dozens of content creation tools available, each with a unique set of (dis)advantages. For many, a "must have" in these tools is the ability to create prototypes and interfaces that can be easily tweaked and pushed to production. Based on these guidlines, this module explores and evaluates four programs: Bohemian Sketch3, Adobe Muse, Adobe Edge Reflow, and Adobe Brackets.
