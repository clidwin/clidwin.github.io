Design Tools: Adobe Edge Reflow

If you do a quick search for "Adobe web design tool", chances are your top result will be Muse, Adobe's branded "web design software" within the Creative Cloud suite. Muse focuses on making a website easy to plan, decent to develop on, fine to preview, and quick to publish to the internet. What makes it an appealing tool is the ability to set different layouts for desktop and mobile web, which helps non-web savy users understand how a layout can change at different sizes. It's also possible to view visual sitemaps for the developing site, which makes it easy to see how different pages and designs relate to one another.

Adobe Edge Reflow CC (Preview) can be found and subscribed to at https://creative.adobe.com/products/reflow

Program Highlights

Some important things to know about Adobe Muse:

  • Provides opportunities to design for multiple layouts, split into the categories of desktop, tablets, and mobile phone.
  • Integrates well with other old Adobe products such as Photoshop and Illustrator.
  • Provides an overview of the design through showing site maps of the content.
  • Does not incorporate fluid responsive design; instead, Muse specifies each size as a separate design.
  • Exported HTML and CSS contains extra code that renders the design less effectively than other technologies.
  • Does not provide an easy way to integrate motion or other animated elements into the static content.

Recommendation

Muse is quickly becoming an outdated product, being replaced by Edge Reflowand the rest of the Edge suite. I would not recommend learning it over the Edge suite, but it's useful to know about as a predecessor to the newer tools.
