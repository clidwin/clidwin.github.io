Design Tools: References

Sketch3 Tutorials is a YouTube channel with video tutorials on the interface and project basics for Bohemian's Sketch3. Each tutorial is roughly five minutes long, making the entire series easy to go through and quick for getting started with the product.
Foundation Adobe Edge Animate: for HTML5, CSS3, and JavaScript Development is book available online through Safari that talks about using Adobe Edge Animate. In particular, there is a chapter on how Edge Animate interacts with Edge Reflow, which is useful for understanding how Reflow and Animate work together within Adobe Edge and the larger Adobe suite of products.
7 Exciting Up-And-Coming Web Technologies evaluates a few different technologies used for web prototyping and development. It provides a good overview of what the new products are in this space and when each one might be a useful addition to your workflow.
6 Coding Playgrounds for Web Developers is an article that compares different tools that run Javascript, HTML, and CSS code online. This set of modules utilizes one of the tools, Codepen, but it is useful to know what else is out there and how the different products allow for code to be written, shared, and interacted with across the web.
