Design Tools: Adobe Edge Reflow

Adobe Edge Reflow is a part of Adobe Edge (a subset in the Creative Cloud suite), and focuses on responsive web design by providing ways for designers to see how changing the size of a screen affects the design. The interface integrates a developer concept of breakpoints and allows users to create points at which the design "breaks" or changes to match a new flow. In a world where it seems every device has a different screen, this tool allows designers to mimic a wide array of devices before pushing the concept to production.

Adobe Edge Reflow CC (Preview) can be found and subscribed to at https://creative.adobe.com/products/reflow

Program Highlights

Adobe Edge Reflow's feature set:

  • Provides a fluid way of looking at responsive design through a draggable widget that changes the displayed width.
  • Integrates well with the Adobe suite, especially other Edge products such as Inspect and Animate.
  • Has a clearly defined purpose of designing responsive content, as opposed to having a lot of disjointed features.
  • Does not allow previewing of other Edge components such as animation.
  • Code editor export creates code with extra tags that is inefficient for code readability, editing, and rendering.
  • Requires each component to be changed independently; there is no concept of a set style guide that can be applied to elements.

Recommendation

For Adobe enthusiasts, Reflow is a great solution to designing with responsiveness in mind. By itself, it does not have the workflow to handle the full scope of web design, but it does responsiveness well, which is the goal of the product. It is well worth looking into if you are already an Adobe subscriber.
