Design Tools: Bohemian Sketch 3

For designing applications that exist within a framework such as iOS or Android, Bohemian's Sketch3 provides useful tools for dragging, dropping, and grouping elements as you design your user interface. The product is focused on making the design part of creating a user interface smooth and easy to use, and unlike some of the more high-end products out there, the cost is under $100 for a regular licence and 50% off for students, which makes it a good investment for those who do not want to commit to a subscription for Adobe suite.

Sketch3 can be found and purchased at http://bohemiancoding.com/sketch/

Program Highlights

Here are some of the features of Sketch3:

  • Provides pre-fabricated elements such as iOS assets to make it easy to start with an existing framework.
  • Create element styles to make it easy to apply and maintain consistency between similar UI components.
  • Makes it easy to duplicate an existing page and copy assets between artboards.
  • Not as versatile as Photoshop in handling advanced image manipulation.
  • Unable to edit CSS for an element within the program itself (must copy the CSS to an external file).
  • Only allows mirroring with iOS mobile phone devices (no Android, smartwatches, etc).

Recommendation

Sketch3 is a great tool if you are looking to design and prototype an application that is using an existing framework such as iOS, Bootstrap, or Material Design and you're using a Mac for development. If you are looking to customize components individually or do some more advanced vector shapes, this tool may not satisfy your needs on its own.
