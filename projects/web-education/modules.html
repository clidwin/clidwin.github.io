Modules

Each module in this site consists of six parts that together compile information from across the internet on how to develop for the web. Through interactive examples using CodePen, you can not only learn about what code generates certain effects, but also modify the examples and "fork" them for use in your own projects. There are also references to how these properties and effects relate to different art tools, providing multiple ways for you to process information about the content and explore some of the latest and more expansive web technologies.

Design Tools
The web industry has multiple tools that can be used to assist with layout, prototyping, animation, and other components of the design and development proccesses. This module looks at four tools and the (dis)advantages of using each one, along with a few tips for getting started with the tool. While this module is not exhaustive, the breadth of products is aimed at giving a glimpse of the different types of software available.
CSS Transforms
In learning CSS, most people discover how to change colors, sizes, and other related properties of different DOM elements. Transforms expand on this topic and are in particular useful for modifying shapes to have different characteristics such as appearing squashed, stretched, or rotated. Usually such characteristics are added using javascript or a related library, but this can also be done just using CSS, as shown through this module.
CSS3 Animations
Like with transforms, CSS3 Animations are considered to be a more advanced topic and are an experimental feature of CSS, but they are not hard to understand or implement if you know how to use vendor prefixes and keyframing concepts. This module assists with providing CSS-only solutions to animating elements within the DOM structure, and expands the possibilities for interactivity within a website.
