Site Resources

6 Coding Playgrounds for Web Developers is an article that compares different tools that run Javascript, HTML, and CSS code online. This set of modules utilizes one of the tools, Codepen, but it is useful to know what else is out there and how the different products allow for code to be written, shared, and interacted with across the web.

Referenced In:About, CSS3 Animations, CSS Transforms

7 Exciting Up-And-Coming Web Technologies evaluates a few different technologies used for web prototyping and development. It provides a good overview of what the new products are in this space and when each one might be a useful addition to your workflow.

Referenced In: Design Tools

CSS3 Animation Cheat Sheet is a fun and interactive way to see a small sample of possible CSS3 animations. The site also has a downloadable CSS file that can be used to apply these animations to your own site.

Referenced In: CSS3 Animations

The CSS3 matrix() Transform for the Mathematically Challenged is a blog post that explains the mathematics of the CSS3 matrix transform. This is useful to know since matrix algebra serves as the foundation for other transforms such as rotate, scale, skew, and translate, and it is also possible to combine multipe transformations into a single matrix function.

Referenced In: Matricies

Comparing the effectiveness of different educational uses of program animations is an article that outlines a study conducted to see how effective animations are as educational tools. It is a technical journal article related to this topic in that the findings provide insight into when animations are useful for educational contexts.

Referenced In: CSS Transforms

Cubic-Bezier is an interactive site that showcases the different timing functions that can be applied to a CSS3 animation, and what values a certain curve has. This is a handy site to test out custom timing functions and generate code to use in in CSS3 animations.

Referenced In: CSS3 Animation Subproperties

Foundation Adobe Edge Animate: for HTML5, CSS3, and JavaScript Development is book available online through Safari that talks about using Adobe Edge Animate. In particular, there is a chapter on how Edge Animate interacts with Edge Reflow, which is useful for understanding how Reflow and Animate work together within Adobe Edge and the larger Adobe suite of products.

Referenced In: Edge Reflow

Jon Duckett's HTML and CSS is one of my favorite web reference books. It is well designed, well written, and well executed. I particularly enjoy how each chapter is structured around both a broad topic and focused example that applies attributes of the chapter. Well worth the read for people who either do not have prior experience in HTML and/or CSS or want a good reference book on these subjects.

Referenced In: CSS Transforms

Keyframe Animation Syntax is a code snippet within the CSS-Tricks website which goes over the basics of CSS3 Animations. This post provides a different way for understanding CSS3 animations by discussing different syntaxes, and the site as a whole is a great resource for both general topics as well as examples of commonly created elements.

Referenced In: Keyframes

Mozilla Developer Network (MDN) is essentially a Wikipedia for web technologies. It is useful for looking up what values a property can have, but what I find especially valuable are its up-to-date listings of what browsers support a certain property.

Referenced In: CSS3 Animations, CSS Transforms

SeeSS: Seeing What I Broke - Visualizing Change Impact of Cascading Style Sheets (CSS) is an article that proposes a methodology to show the scope of elements effected by a CSS property change. The proposal is based on the observation that a difficult part of writing CSS is seeing how one property change affects the site as a whole. It is a technical journal article, but has interesting insights on where web developer tools are lacking and how the proposed SeeSS system is a viable option.

Referenced In: CSS Transforms

Sketch3 Tutorials is a YouTube channel with video tutorials on the interface and project basics for Bohemian's Sketch3. Each tutorial is roughly five minutes long, making the entire series easy to go through and quick for getting started with the product.

Referenced In: Sketch3

Squash, Stretch and Bounce With CSS3 Animations is blog post showcasing how to use the general animation principle squash and stretch to make a CSS3 animation looking more realistic.

Referenced In: CSS3 Animations

Vendor Prefixes is a webpage that describes what vendor prefixes are, how they are used, and what browsers need which prefixes in order to enable experimental CSS features.

Referenced In: CSS3 Animation Prefixes

W3C: CSS Animations is the official documentation on CSS Animations. The provider, W3C, is the World Wide Web Consortium, which is the group that develops standards for the web. It is a more technical source than some of the other sites, but is a great resource to see what the current standards are and what future standards are being drafted.

Referenced In: CSS Animations
