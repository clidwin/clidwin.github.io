Developing the Information Aggregator

Information about the development process

  • Looked at OpenData, RunTracker, and a few other programs to record longitudinal data (make a chart of the features of these programs and decribe why I went the custom route).
  • Prototyping app to record data using the tutorial Making Your App Location-Aware
  • Decided to use AppEngine as a database storage facility and multi-client server.
  • Created an AppEngine test application using “App Engine Java Endpoints Module” Template (it says hello to the user)
  • Store sample data into AppEngine to test ability to store and retrieve data.
  • Motion-based tracking instead of location-based (update with movement rather than from a set time interval)
