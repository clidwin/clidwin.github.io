This page contains the (growing) list of resources used in developing this thesis. The intent is to build in the ability to filter by source type, where it contributes to the thesis, and alphabetically.

Enchanted Objects
By David Rose
Annotation Source
Rose not only explores how we might interact with technologies in the future, but also what makes an object "enchanted" and memorable. In particular, this source gives insight on how ambient displays can be used to better understand data.
Overview Source
the functional art
By Alberto Cairo
Annotation Source
How do we transform data from its raw forms into something more visually appealing? Cairo both draws from his own experiences and interviews experts in the field of information graphics, and provides insights on the process of creating information-driven visualizations.
Overview Source
Thinking,
Fast and Slow
By Daniel Kahneman
Annotation Source
A source listed in Enchanted Objects, this book details the concept of a two-system brain and how peope both consciously and subconsciously process information. Kahneman also gives suggestions on when we should (and should not) rely on these different systems.
